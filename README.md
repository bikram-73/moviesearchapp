# MovieFlix - Movie Explorer 🎬

MovieFlix is a responsive, feature-rich web application that allows users to discover, search, and filter movies. Built with vanilla JavaScript, HTML, and CSS, it dynamically fetches and displays movie data using the TMDB (The Movie Database) API.

## ✨ Features

* **Trending Carousel:** A horizontally scrollable carousel displaying the top trending movies of the week.
* **Search Functionality:** Real-time search to find specific movies by title (with built-in debouncing to optimize API calls).
* **Advanced Filtering & Sorting:**
  * Filter by **Genre** (Action, Comedy, Drama, etc.)
  * Filter by **Release Year** (1990 to Current Year)
  * Sort by **Popularity, Rating, Release Date, or Alphabetical (A-Z)**
* **Discover Section:** Displays a randomized grid of movies on initial load to help users discover new content.
* **Responsive Design:** Fully mobile-friendly layout with CSS Grid and Flexbox.

## 🛠️ Technologies Used

* **HTML5** - Semantic structure
* **CSS3** - Custom styling, UI/UX, animations, and responsive media queries
* **JavaScript (ES6+)** - DOM manipulation, Event handling, Async/Await, and Object-Oriented structure (`class MovieExplorer`)
* **TMDB API** - Backend data source for movies, genres, and images

## 🚀 Getting Started

To run this project locally, you will need to set up a configuration file for the TMDB API key.

### Prerequisites
* A modern web browser (Chrome, Firefox, Safari, Edge)
